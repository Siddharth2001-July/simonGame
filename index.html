How the Simon game works





  • The game shows the colors in the sequence. (Ex - blue)

  • The user clicks on the button.

  • Then game shows the next colour. (Ex - red)

  • The user has to remember the sequence is blue, red and so on.

  • Then enter the Sequence from starting (Ex - blue, red ...) at every level

  • If you messes up the sequence, then the game ends
